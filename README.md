# 裂变贺卡
用户上传图片后，通过鼠标/触摸滑动分裂圆点来逐步揭示图片内容，最终生成个性化贺卡

核心功能是用户上传图片后，通过鼠标/触摸滑动分裂圆点来逐步揭示图片内容，最终生成个性化贺卡。

## 核心功能需求

1. 图片上传与裁剪

- 支持上传本地图片

- 提供可视化裁剪工具，允许用户调整裁剪区域

- 裁剪完成后将图片转换为Canvas可处理的格式

2. 圆点分裂效果

- 初始状态为一个大圆点，覆盖整个画布

- 鼠标/触摸滑动时，接触到的圆点分裂为4个小圆点

- 圆点分裂后保持原图片的色彩信息

- 圆点具有立体感（高光和阴影效果）

- 最小分裂半径限制，避免无限分裂

3. 文本编辑

- 标题和描述文本支持编辑（contenteditable）

- 默认文本为"致最特别的你"和"滑动鼠标，发现藏在圆点里的秘密"

4. 画布控制

- 一键重置画布功能，恢复初始状态

- 当图片加载失败时，自动生成渐变纹理作为兜底

5. 视觉设计

- 自定义光标（SVG实现）

- 卡片式布局，包含阴影和圆角

- 渐变背景

- 响应式设计，支持鼠标和触摸操作

## 技术实现要求

1. 技术栈

- 纯前端实现：HTML5, CSS3, JavaScript (原生)

- 核心技术：Canvas API, FileReader API, DOM操作

2. 代码结构

- 面向对象设计：Circle类封装圆点的属性和方法

- 模块化函数：初始化、渲染、交互处理等

- 性能优化：防抖处理，避免频繁计算

3. 交互逻辑

- 鼠标/触摸事件处理

- 裁剪框拖拽和 resize 功能

- 坐标转换，确保触摸/鼠标位置的准确性

4. 兼容性

- 支持现代浏览器

- 响应式适配不同设备尺寸

## 界面设计

1. 布局结构

- 居中卡片布局

- 顶部为可编辑标题和描述

- 中间为Canvas画布

- 底部为控制按钮（上传照片、重置画布）

2. 视觉风格

- 现代简约风格

- 柔和的颜色方案

- 精致的阴影和圆角效果

- 流畅的动画过渡
